## 🚀 下一步操作

### 1. 部署隐私政策（必须）
```bash
# 选项 A: 使用 GitHub Pages
1. 将 AppStoreAssets/privacy-policy.md 转换为 HTML
2. 提交到 GitHub 仓库
3. 在仓库设置中启用 GitHub Pages
4. 获取 URL（如：https://longhaiqwe.github.io/mashangji/privacy-policy.html）

# 选项 B: 使用 Vercel（推荐，更简单）
1. 在 Vercel 中新建项目
2. 上传 privacy-policy.md（或 HTML 版本）
3. 部署并获取 URL
```

### 2. 创建支持页面（推荐）
创建一个简单的 HTML 页面，包含：
- App 介绍
- 使用教程
- 常见问题
- 联系方式

### 3. 在 Xcode 中准备构建
```bash
# 1. 构建生产版本
npm run build

# 2. 同步到 iOS
npm run cap:sync

# 3. 打开 Xcode
npm run cap:open

# 4. 在 Xcode 中：
#    - 选择 Generic iOS Device
#    - 检查签名配置
#    - Product → Archive
#    - Upload to App Store
```

### 4. 在 App Store Connect 中配置
1. 上传所有截图
2. 填写所有文案
3. 添加隐私政策 URL
4. 选择构建版本
5. 填写审核信息
6. 提交审核

3. **测试账号问题**
