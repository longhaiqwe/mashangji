# 🀄️ 马上记 (Ma Shang Ji)

**你的 AI 智能牌局管家 | 语音秒记，运势尽在掌握**

还在用备忘录或 Excel 记流水账？是时候升级你的“战绩”管理方式了！**马上记** 是一款专为棋牌和聚会游戏爱好者打造的智能记账工具。我们利用最前沿的 AI 技术，让你只需动动嘴，就能轻松记录每一场风云变幻。

### 🌟 核心亮点

#### 🗣️ **AI 语音秒记，解放双手**
告别繁琐的手动输入！只需**长按底部“+”号**，直接说出战绩，DeepSeek AI 智能引擎将自动识别**时间、金额、输赢、圈子**和**备注**，甚至支持批量录入。
> *"昨天在同事圈打麻将赢了 200，周五和老张斗地主输了50"* —— 一句话，多笔账单瞬间搞定。

#### 🔮 **运势分析 & 智能复盘**
不仅仅是记账，更是你的军师。
- **本周运势**：根据近期战绩，AI 实时生成运势签文（“气势如虹”、“扭转乾坤”），助你把握最佳战机。
- **月度复盘**：自动生成“开局不利”、“强劲反弹”等月度总结，一眼看穿盈亏趋势。

#### 📊 **专业数据看板**
- **多维统计**：总场次、胜率、最大连红、最大回撤，数据一目了然。
- **图表可视化**：红绿分明的盈亏趋势图（红赢绿输），直观展示你的“战力”曲线。

#### 👥 **圈子管理**
同事、家人、牌友……不同圈子独立核算，账目清晰不混淆。

#### 🎨 **极致 UI 体验**
- **玻璃拟态设计**：现代感十足的磨砂玻璃界面。
- **多主题切换**：内置“发财红”、“新春限定”、“极简黑”等多种主题，随心切换，不仅要赢，还要赢得漂亮！

### 🛡️ 隐私与安全
- **本地优先**：数据安全存储，你的战绩只有你知道。
- **无广告打扰**：纯净体验，专注于每一次记录。

## 🛠 Developer Guide

### Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `VITE_SILICONFLOW_API_KEY` in [.env](.env) to your SiliconFlow API key
3. Run the app:
   `npm run dev`
