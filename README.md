# 🀄️ 马上记 (Ma Shang Ji)

**你的 AI 智能牌局管家 | 语音秒记，运势尽在掌握**

还在用备忘录或 Excel 记流水账？**马上记** 利用前沿 AI 技术，让你只需动动嘴，就能轻松管理每一场“战绩”。

### 🌟 核心亮点

1.  🗣️ **AI 语音秒记**：长按底部“+”号说话，智能识别时间、金额与输赢，一句话搞定多笔账单。
2.  🔮 **运势智能复盘**：实时生成运势签文，自动分析月度盈亏趋势，助你把握最佳战机。
3.  📊 **专业数据看板**：胜率、盈亏、连红一目了然，红赢绿输图表直观展示“战力”曲线。
4.  👥 **多圈子独立账本**：同事、牌友、家人圈子隔离，多人对局账目清晰，拒绝糊涂账。
5.  🎨 **极致个性体验**：玻璃拟态设计，内置“发财红”、“新春”等多款主题，数据本地存储更安全。

---

## 🛠 Developer Guide

### Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `VITE_SILICONFLOW_API_KEY` in [.env](.env) to your SiliconFlow API key
3. Run the app:
   `npm run dev`
